# Enclosures And Covers Shortlist

This shortlist is the first working family set for the reference-library intake process.
It is a planning artifact, not canon.

## Selection Rule

Prefer candidates that:

- reuse landed `shell` capability
- have clear envelope and wall dimensions
- expose one new idea only
- produce a useful workflow follow-on

## Candidates

### Flanged Junction Box Body

- utility: high
- clarity: high
- likely novelty: low
- current fit: strong
- notes:
  - good first slice
  - reuses shell plus joined exterior features
  - naturally leads to a lid follow-on

### Friction-Fit Lid With Stepped Lip

- utility: high
- clarity: medium
- likely novelty: medium
- current fit: medium
- notes:
  - better as a second slice after a matching box body exists
  - likely introduces explicit fit-gap handling

### Terminal Block Splash Shield

- utility: medium
- clarity: medium
- likely novelty: medium
- current fit: medium
- notes:
  - may force thin-wall and flex assumptions too early
  - keep shortlisted but not first

## Recommended Order

1. flanged junction box body
2. friction-fit lid with stepped lip
3. terminal block splash shield

## Next Step

Create normalized part-spec files for the 3 candidates above, then score them against the implementation rubric.
