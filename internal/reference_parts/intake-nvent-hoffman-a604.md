## Notes

- This is a standard industrial junction box with well-defined envelope dimensions.
- The continuous hinge with clamps closure style is distinct from screw-cover designs.
- Wall mounting is via integrated welded brackets (not separate flange ears).
- Multiple variants exist: A604CH (Type 12), A604CHNF (Type 4), A604LP (lift-off cover), A604SC (screw cover).
- Selected A604CH as the primary reference (most common Type 12 variant).
