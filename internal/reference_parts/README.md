# Reference Part Collection

This directory holds normalized dimensional specifications and design challenges for real-world utility-part families. These parts serve as benchmark targets for testing ParamAItric's capability to generate accurate, fit-critical replacements.

## Part Selection Criteria

- **FDM Printable:** Avoid parts requiring microscopic tolerances or extreme material strength not possible in standard desktop 3D printing.
- **Mathematically Definable:** Prefer parts defined by clear, discrete parameters (bolt hole patterns, flanges, standard tapers).
- **Utility Focus:** Focus on brackets, adapters, spacers, knobs, handles, and basic enclosures.

## Reference Input Rule

This directory is for normalized part-family specifications and workflow-relevant dimensional notes, not raw reference dumps.

Use it to capture only the minimum geometry, interfaces, and constraints needed to drive a reproducible workflow slice.
