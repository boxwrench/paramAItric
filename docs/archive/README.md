# Docs Archive

This folder holds tracked historical documentation that is still worth keeping but should no longer be treated as current repo guidance.

Current rule:

- active product, roadmap, and research direction stay in `README.md`, `docs/`, and the reviewed `internal/research/` bundles
- historical migration writeups, superseded handoffs, and retired planning snapshots belong here
- do not cite archived docs as current state without checking the newer canonical docs first
