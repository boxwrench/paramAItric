# ParamAItric

CAD with questionable intelligence.

ParamAItric connects AI agents to Fusion 360 so they can design parts, automate modeling workflows, and occasionally create something accidentally brilliant.

Or at least something printable.

This project started as an experiment to see if an LLM could drive Fusion 360 through the Python API and Model Context Protocol (MCP).

The result is a system where AI can:

• create sketches  
• extrude features  
• generate mechanical parts  
• automate CAD workflows  
• export printable models  

And sometimes add one constraint too many.

Serious CAD engineers may roll their eyes.

That's fair.
